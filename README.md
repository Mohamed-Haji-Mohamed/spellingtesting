This Website is designed to help English learners, children and anyone who wants to Improve or test their Spelling skills.

## 🚀 Features

- ✅ Simple and responsive design
- 🔤 Multiple Choice questions
- 🔄 Restart quiz option
- ⏭️ Next option
- 📱 Mobile-friendly layout

- HTML5
- CSS3
- JavaScript
